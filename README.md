# Beyond Themes: Hybrid Arch Linux Setup

> **A raw, custom-stitched environment. No noise. Just the flow.**

---

This repository is a manual fusion of different aesthetics, adjusted by hand for a focused workflow. It’s not a standard theme; it’s a mix of what works.

## 🎨 Visuals
* **Colors:** A custom blend of **Tokyo Night** depths (`#1a1b26`) and **Catppuccin Mocha** accents.
* **Borders:** Color-coded module separation (Cyan / Mauve / Peach).
* **Fonts:** Roboto Nerd Font (UI) & JetBrains Mono (Code).

## 🛠 Tech Stack
* **WM:** `Hyprland`
* **Bar:** `Waybar` (Modular layout)
* **Terminal:** `Kitty` (0.75 opacity)
* **Launcher:** `Wofi`

## 🚀 Usage
This is a **Template Repository**. 

1. Use this template to create your own repo.
2. Copy files from `configs/` to `~/.config/`.
3. Tweak the variables in CSS to adjust the colors.
