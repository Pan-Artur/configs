# 🧿 Abreaction: The Hybrid Shell

> **"A soul-driven Arch Linux configuration. Clean code. Clean mind."**

---

**Abreaction** is an eclectic, high-performance configuration template for Arch Linux. It’s not a "by the book" theme—it's a raw, manual fusion of the best aesthetics in the Linux world, tuned for focus and flow.

## 🌌 The Concept
This config represents the transition from chaos to clarity. It’s a **"Frankenstein"** setup in the best sense: taking the soul of **Catppuccin**, the depth of **Tokyo Night**, and welding them together with custom neon accents.

## 🎨 Visual Architecture
* **The Core:** A sophisticated mix of *Catppuccin Mocha* surfaces and *Tokyo Night* dark depths (`#1a1b26`).
* **The Accents:** Sharp, color-coded borders for UI modules:
    * `Cyan` for focus.
    * `Mauve` for logic.
    * `Peach` for alerts.
* **Typography:** Built on **Roboto Nerd Font** for a clean interface and **JetBrains Mono** for an uncompromising coding experience.

## 🛠 Tech Stack
* **WM:** `Hyprland` (Fluid animations & Tiling logic)
* **Bar:** `Waybar` (Multi-color modular architecture)
* **Terminal:** `Kitty` (0.75 transparency for that "floating" feel)
* **Launcher:** `Wofi` (Minimalist blue-themed execution)
* **Music:** Integrated Spotify status module with signature branding.

## 🚀 Usage as Template
This is a **Template Repository**. It’s designed to be gutted, modified, and rebuilt:

1.  Click **"Use this template"**.
2.  Deploy files to `~/.config/`.
3.  Tweak the `@define-color` variables to match your own frequency.

## 🧿 Philosophy
If **Katarsis** is the destination, **Abreaction** is the engine. Every line of this CSS is a release of digital tension. No bloat. No noise. Just the shell and your thoughts.
